Website usability and accessibility

Website accessibility to all types of visitors is important to me and that is why I have tried my best to design the website for people with all disabilities and impairments, and for people on all internet devices and browsers.

Images

I have provided alternative text to images, so that visually impaired users and people using text-only browsers do not miss out on information which images on the site provide.

Navigation

The website uses simple text links for its navigation, with no javascript.

Links

All links on this site open in the same window, so your browser's 'back' button will always work. All links are accurate and in working order. Links to external sites will also be periodically checked.

Access Keys

Access keys are combinations of keys that can be used to navigate through the site without the use of a mouse. For the benefit of users who find using a mouse difficult, I have provided access keys for the following pages:

  • 0: Home

  • 1: About Myself

  • 2: Education

  • 3: Courses

  • 4: Achievements

  • 5: Documents

  • 6: Hobbies

  • 7: Usability Guide

What you need to do to use access keys depends on your browser:

  • Internet Explorer(PC)- Hold Alt, press access key, release, press Enter
  • Internet Explorer, Safari(Mac)- Hold Ctrl, press access key
  • Google Chrome- Hold alt, press access key
  • Firefox(Mac)- Hold Ctrl, press access key
  • Firefox(PC)- Hold Shift and Alt, press access key
  • Opera- Hold Shift, press Escape, release, press access key

NB: The Access Key is the number shown above in bold.

Document Structure

I have designed the site with the aim of achieving maximum clarity and consistency throughout the site in order to make it easy and quick to find your way around the site. The underlying code of each page has a linear structure and HTML headings have been used so that non-visual browsers(screen readers) can read each page in a logical sequence. Every page begins with page title and the main navigation menu. The content follows.

Browser Compatibility

This website uses cascading style sheets(css) for layout. A less complex style sheet is automatically provided for browsers which do not fully support css. For users with these browsers, some decorative elements of the site will not be visible. People with browsers which have no support for css at all will be viewing the unstyled version of this website. However, users with these older browsers still have full access to all the site's content
